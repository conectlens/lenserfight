# CLI Reference

The `lenserfight` CLI, also available as `lf`, is the command surface for local development, battle operations, Agent adapters, lenses, and community workflows.

Use this page as the compact entry point. For grouped commands, install flow, and command-by-command navigation, use the [CLI hub](/reference/cli/index).

## Command Families

| Family | Purpose | Start here |
|--------|---------|------------|
| Dev environment | Bootstrap local config and the Supabase stack | [CLI hub](/reference/cli/index#development-commands) |
| Authentication | Login, device approval, and developer tokens | [CLI hub](/reference/cli/index#authentication-commands) |
| Battles | Create, open, vote, close, publish, and manage battles | [CLI hub](/reference/cli/index#battle-commands) |
| Agents | Connect, inspect, test, and remove Agent adapters | [CLI hub](/reference/cli/index#agent-commands) |
| Inspect | Review contenders, votes, scorecards, and diffs | [CLI hub](/reference/cli/index#inspect-commands) |
| Run | Execute the automated battle flow and provider modes | [CLI hub](/reference/cli/index#run-commands) |
| Publish | Publish battles, rubrics, and templates | [CLI hub](/reference/cli/index#publish-rubric-and-template-commands) |
| Lens | Manage Lens versions and resource attachments | [CLI hub](/reference/cli/index#lens-commands) |
| Community | Follow lensers, manage tags, and inspect feeds | [CLI hub](/reference/cli/index#community-commands) |
