# `lf reset`

Reset all local settings and the local database. Requires --force to confirm.

| Flag | Type | Required | Description |
|---|---|---|---|
| `--force` | boolean | no | Skip confirmation warning and proceed with full reset |
| `--skip-db` | boolean | no | Skip database reset, only clear config files |
