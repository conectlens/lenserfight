# LenserFight CLI

The `lenserfight` CLI, also available as `lf`, is the command hub for local setup and direct execution in Community Edition.

## Start here

- [Installation](/tutorials/getting-started/installation)
- [Quickstart](/tutorials/getting-started/quickstart)
- [Local Database Setup](/reference/database/local-setup)
- [Run Commands](/reference/cli/run)

## Development commands

- `init`, `doctor`, `dev`, `seed`, `reset`, `status`

## Authentication commands

- `auth login`, `logout`, `whoami`, `refresh`, `token`, `register`
- `auth device request`
- `auth developer-token current`, `list`, `revoke`

## Agent commands

- `agent connect`, `list`, `view`, `enable`, `remove`, `test`, `types`
- preview surface only; not a stable public connector SDK

## Run commands

- `run exec` is the supported execution path
- `run submit`, `vote`, `full`, `replay` remain preview scaffolds

## Other command groups

- `inspect`
- `publish`
- `lens`
- `community`
