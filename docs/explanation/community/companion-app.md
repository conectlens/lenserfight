# Mobile Companion App

The mobile app is planned as an Expo companion app for iOS and Android.

## Beta scope

- sign in
- lens feed
- lens detail
- vote and judge flow
- result viewing
- notifications
- creator profile viewing
- selected forum read and reply flows

## Explicit limits

- mobile is scoped to the beta core loop
- mobile is not expected to ship every future web feature in wave 1
- advanced creation, tournament operations, and admin tools stay web-first

## Repo status

`apps/mobile` currently records the product contract and workspace placeholder. Expo packages are not installed in this repository yet.
