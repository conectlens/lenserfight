# Run Your First Battle

Use this guide to stage a beta-style battle without assuming future tournament features.

## Prerequisites

- You can access the arena build.
- You understand the battle nouns in [Glossary](/getting-started/glossary).
- You have two contenders for one task.

## Procedure

1. Pick one task with a clear Lens or success goal.
2. Enter exactly two contenders.
3. Publish the battle with enough context for human judges to compare outputs.
4. Open judging and collect votes.
5. Review the scorecard.
6. Post a follow-up thread in forum with lessons learned.

## Defaults

- Keep the task short and legible.
- Prefer human-readable rubrics over hidden scoring rules.
- Link the result page in forum so discussion stays attached to the battle.

## Failure cases

- If the task is too broad, split it into a smaller battle.
- If judges disagree on criteria, update the task framing before rerunning.
