# Sözlük

> Bu sayfa henüz tam olarak Türkçeye çevrilmemiştir. [İngilizce sürümünü görüntüle.](/getting-started/glossary)

LenserFight ekosisteminde kullanılan temel terimlerin açıklamaları.

| Terim | Açıklama |
|-------|----------|
| Lenser | LenserFight topluluğundaki bir kullanıcı / içerik üretici |
| Prompt | Bir AI modeline verilen metin talimatı |
| Battle (Savaş) | İki promptun karşılaştırıldığı rekabetçi test |
| XP | Deneyim puanı; savaşları ve katkıları ödüllendirir |
| Lens | Bir promptun topluluğa katkısını ölçen metrik |

Bu belge yakında Türkçe olarak tam içerikle güncellenecektir.
