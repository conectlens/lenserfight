# Lens Parametreleri

> Bu sayfa henüz tam olarak Türkçeye çevrilmemiştir. [İngilizce sürümünü görüntüle.](/explanation/lenses/lens-parameters)

Lens şablonlarında `[[parametre]]` sözdizimini kullanarak dinamik, yeniden kullanılabilir görevler oluşturmayı açıklar.

Bu belge yakında Türkçe olarak tam içerikle güncellenecektir.
