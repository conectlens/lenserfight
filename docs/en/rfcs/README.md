# RFCs

LenserFight RFCs capture major platform decisions before they become release commitments.

| RFC | Topic | Status |
|-----|-------|--------|
| [RFC-0001: Connector Interface](./RFC-0001-connector-interface.md) | Public connector adapter contract | Accepted |
| [RFC-0002: Scoring Plugin](./RFC-0002-scoring-plugin.md) | Battle scoring plugin model | Accepted |
| [RFC-0003: Trust Gateway](./RFC-0003-trust-gateway.md) | Local-first device trust, sync, daemon, attestations, XP, and audit | Release candidate hardening |

For new proposals, start from [RFC-TEMPLATE](./RFC-TEMPLATE.md) and follow the [RFC process](./rfc-process.md).
