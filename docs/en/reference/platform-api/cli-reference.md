# CLI Reference

The `lenserfight` CLI, also available as `lf`, is the command surface for Community Edition local setup and direct execution.

Use this page as the compact entry point. For grouped commands and current scope notes, use the [CLI hub](/en/reference/cli/index).

## Command families

| Family | Purpose | Start here |
|--------|---------|------------|
| Dev environment | Bootstrap local config and the Supabase stack | [CLI hub](/en/reference/cli/index) |
| Authentication | Login, device approval, and developer tokens | [CLI auth](/en/reference/cli/auth) |
| Run | Direct execution and preview run scaffolds | [Run Commands](/en/reference/cli/run) |
| Agents | Preview agent record management | [Agent Commands](/en/reference/cli/agent) |
| Lens | Manage Lens versions and resource attachments | [Lens Commands](/en/reference/cli/lens) |
| Community | Follow lensers, manage tags, and inspect feeds | [Community Commands](/en/reference/cli/community) |

## Scope note

`lf run exec` is the primary supported execution path in the current OSS beta. Broader automation flows remain preview-only.
