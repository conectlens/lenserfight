# `lf connect`

Subscribe to, disconnect from, list, or sync public lenses and workflows.

| Flag | Type | Required | Description |
|---|---|---|---|
| `<target>` | positional | no | Lens or workflow slug to connect (omit when using --list or --sync) |
| `--disconnect` | string | no | Disconnect from a lens or workflow slug |
| `--list` | boolean | no | List all connected lenses and workflows |
| `--sync` | boolean | no | Pull latest versions for all connected lenses and workflows |
| `--json` | boolean | no | Output as JSON (with --list) |
