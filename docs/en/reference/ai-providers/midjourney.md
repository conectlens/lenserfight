# Midjourney

Midjourney appears in the LenserFight provider catalog for comparison and workflow planning purposes. Midjourney 7 is listed in the default model registry but is not yet active in the platform execution surface; it must be enabled via BYOK credentials.

## Support tier

`catalog_only` — The provider appears in discovery, planning, and showroom UIs but is not executed directly without BYOK configuration.

## Usage notes

- To use Midjourney 7 in a lens or battle, configure your Midjourney API credentials under Settings → Partner Accounts and select the model in the lens version's `model_id` field.
- Midjourney is known for highly stylised, artistic output — it tends to diverge from photorealistic accuracy; factor this into battle judging criteria.
