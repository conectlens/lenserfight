# Wave 2 Contributor Plan

Wave 2 is the second cohort of contributors invited to help build LenserFight Community Edition. It follows the early-access period and is focused on production hardening, ecosystem breadth, and OSS launch readiness.

---

## Goals

- Expand adapter and connector coverage across popular AI providers
- Improve test coverage, especially integration and e2e scenarios
- Contribute translations across the 10 supported locales
- Harden battle templates and workflow tooling for the v1.0 release window

---

## How to join

Open a thread in [GitHub Discussions — "Wave 2 Interest"](https://github.com/conectlens/lenserfight/discussions) with a short description of what you want to work on. The core team reviews weekly.

Priority is given to contributors who have:
- At least one merged pull request in Wave 1 or early access
- Demonstrated expertise in AI/LLM tooling, Postgres, or frontend (React/Vue)
- Active presence in the community discussions

---

## Incentives

| Contribution type | Recognition |
|-------------------|-------------|
| Merged adapter (new provider) | Adapter author credit in docs + release notes |
| Merged translation (full locale) | Locale maintainer credit |
| Bug fix with regression test | Shout-out in release notes |
| Template authoring (3+ templates) | Template author badge on profile |
