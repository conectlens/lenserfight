# Community Early Access

LenserFight Community Edition is in early access. This page explains who can participate, what is available, and how to submit feedback.

---

## Request access

Submit an access request via [GitHub Discussions — "Early Access"](https://github.com/conectlens/lenserfight/discussions). The core team reviews requests within 5 business days.

Participants are prioritised if they are:
- Active contributors to an OSS AI/LLM project (verifiable GitHub activity)
- Existing Chainabit users who have run at least one workflow
- Maintainers of prompt engineering or AI agent communities
- Referred by an existing early-access participant

## Feedback channels

| Type | Channel |
|------|---------|
| Bug reports | GitHub Issues — `bug` label; adapter bugs use `adapter-bug` |
| Feature requests | GitHub Discussions — "Ideas" category |
| Schema proposals | GitHub Issues — `schema-proposal` label with migration draft |
| General discussion | GitHub Discussions — "General" category |
| Moderation escalation | `moderation@lenserfight.org` |

Do not post credentials, API keys, or personal data in public channels.
