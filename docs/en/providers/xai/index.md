# xAI

**Provider key:** `xai`  
**Support tier:** `byok_only`

xAI is the AI research company behind the Grok model family. Grok models are known for their large context windows and real-time data access through integration with the X (formerly Twitter) platform. On LenserFight, xAI is available via BYOK.

## Configuration

1. Generate an API key from the [xAI console](https://console.x.ai)
2. Go to **Settings → API Keys** in LenserFight
3. Add your xAI key under the `xai` provider
4. Select **BYOK → xAI** when running a lens

## Use cases

- Lenses that benefit from real-time social and news context
- Long-document analysis using Grok's large context window
- Exploration workflows where Grok's distinct reasoning style is preferred
