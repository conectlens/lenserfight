# ElevenLabs v4

**Key:** `elevenlabs-v4`  
**Provider:** [ElevenLabs](/en/providers/elevenlabs/)  
**Support:** `runnable`

## Capabilities

audio_generation

## Specifications

| Field | Value |
|-------|-------|
| Input modalities | text |
| Output modalities | audio |
| Execution pattern | Async task-poll |

## When to use

High-quality text-to-speech with voice cloning. Use in lenses that generate narration, accessibility audio, or podcast-style audio from text content. Async: the execution engine polls until the audio file is ready.
