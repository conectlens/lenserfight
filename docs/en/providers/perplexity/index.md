# Perplexity

**Provider key:** `perplexity`  
**Support tier:** `byok_only`

Perplexity AI offers search-augmented language models that can query the web in real time before generating a response. Ideal for research lenses that need up-to-date information beyond a training cutoff.

## Configuration

1. Create an account at [perplexity.ai](https://www.perplexity.ai) and generate an API key
2. Go to **Settings → API Keys** in LenserFight
3. Add your Perplexity key under the `perplexity` provider
4. Select **BYOK → Perplexity** when running a lens

## Use cases

- Research assistant lenses that pull current facts
- News summarisation with live web search
- Competitive intelligence workflows that need recent data
