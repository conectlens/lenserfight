## The LenserFight loop

```
You write a Lens  →  Two contenders respond  →  Community votes
         ↑                                              ↓
    XP & reputation  ←  Winner earns rank  ←  Results published
```

Every iteration makes the arena smarter. Every vote shapes what "good" means.
