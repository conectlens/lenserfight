# Contributors

Thank you to everyone who contributes to LenserFight.

For contribution guidance, start with [Contributing](/community/contributing). This page focuses only on recognition.

## How Contributors Are Recognized

- This page is auto-synced from the GitHub contributor graph for `connectlens/lenserfight-web`.
- The generated section below recognizes GitHub-account contributors associated with repository commits.
- Review-only, issue-only, design-only, and other non-commit contributions are valuable, but they are not added here automatically unless GitHub also counts them as repository contributors.

## Ways to Contribute (Not Just Code)

- Improve documentation in `docs/` (clarify setup steps, add examples, fix broken links).
- Report bugs with minimal reproduction steps.
- Review PRs (especially docs and small fixes).
- Propose architecture improvements as issues with tradeoffs and constraints.
