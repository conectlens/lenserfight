# LenserFight Mobile

This folder reserves the Expo companion app scope for the April-May 2026 beta.

## Planned beta loop

- sign in
- browse the battle feed
- open battle detail and result pages
- vote or judge
- receive notifications
- view creator profiles
- read and reply in selected forum flows

## Current repo status

Expo dependencies are not installed in this workspace yet. The mobile contract is documented in `docs/mobile/companion-app.md`.
