# Security Policy

Do not report security vulnerabilities in public GitHub issues.

This root file exists for GitHub community health features. The canonical security guide lives at `docs/how-to/contributors/security.md`.

## How to report privately

Preferred: GitHub Security Advisories.

1. Open the repository on GitHub.
2. Go to **Security**.
3. Choose **Advisories**.
4. Select **Report a vulnerability**.

If you cannot access the Security tab, do not disclose publicly. Ask maintainers for a safe private reporting path first.
